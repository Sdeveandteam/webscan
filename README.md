# MiniWebScan

A lightweight web vulnerability scanner built in Python for security testing authorized targets.

**Author:** sdev

## Features
- Security headers analysis with scoring system
- Directory brute force with multithreading
- HTTPS support with SSL verification toggle
- Clean CLI output using Rich
- Export results to file

## Installation
```bash
pip install -r requirements.txt
